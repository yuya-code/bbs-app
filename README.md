# フォルダ解説 src
app
フロント側のファイルが入ってる。
ページやコンポーネント

actions
サーバーアクション
DBの接続やDB内にデータを作成

entities
DBのテーブルの構造をtypescriptで書いてる

migrations
entitiesのtypescriptで定義したデータベースの構造を実際に反映させる処理みたいなものがある

utils
サーバーとフロント分離が難しいものが入ってる



# Proxyとは（プロキシ）
ユーザーがページにアクセスする前に任意の処理を走らせる機能。
例：ユーザーがあるページにアクセスする前にログインしているのかどうか。
ログインしていないなら、ログインページにリダイレクトするというチェックなりリダイレクト処理だったりを走らせる際によく使われる。

Next.jsではsrc直下のproxy.tsがある場合は自動的にそれを認識してユーザーがページにアクセスする前に走らせるようになる。
また、そのファイルの中でconfigのオブジェクトがある場合、PROXYの設定として認識される
